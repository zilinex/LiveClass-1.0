LiveClass: Interactive Classroom Platform

Author: Mehdi Adibi

Date: January 2025

Version: 1.0

Overview

LiveClass is an interactive classroom platform designed to enhance classroom engagement by enabling real-time question and answer sessions. It is not limited to traditional classroom settings but can be applied in a variety of academic environments including lectures, seminars, and even live data collection for research from large crowds. The platform allows teachers to present questions to students, track responses, and display real-time statistics. Students can participate by scanning a QR code and selecting responses. This platform is valuable for both educational purposes and research data collection, with applications in both small and large-scale environments.

Technologies Used

  • Flask (Python web framework)
  • Python (Programming Language)
  • HTML (Frontend for displaying questions and responses)
  • CSS (Styling the interface)
  • JavaScript (Client-side functionality)

Installation Instructions

1. Install Python

Ensure you have Python 3.7+ installed on your system. If not, you can download and install it from here.

2. Install Dependencies

Run the following command to install the necessary dependencies:

pip install flask

3. Running the Application

To run the app, use the following command syntax:

python app.py

Example:

python app.py questions.json

This will start the Flask web server and allow you to access the interactive classroom platform at http://127.0.0.1:5000/.

4. Create a questions.json File

Your questions.json file should follow this structure:

{
  "questions": [
    {
      "question": "Do you find interactive classes beneficial?",
      "options": ["Yes", "No"]
    },
    {
      "question": "What is your preferred learning method?",
      "options": ["Visual", "Auditory", "Kinesthetic"]
    }
  ]
}

Each question should include:

  • A question field for the text of the question.
  • An options array with the available answer choices.

Using Excel to Create the JSON File

1. Create a table with two columns: one for the question and the other for the options.

2. Fill in the rows with your questions and their corresponding options.

3. Save the file as a .csv file.

4. Use an online tool like CSV to JSON Converter to convert your .csv file into a questions.json file.

Excel Example for Questions and Options
Question Options
Do you find interactive classes beneficial? Yes, No
What is your preferred learning method? Visual, Auditory, Kinesthetic
Do you prefer group work over individual tasks? Yes, No

Using Google Sheets to Create the JSON File

1. Create a Google Sheets document with two columns: one for the question and one for the options.

2. Fill in the rows with the questions and their corresponding options.

3. After entering all the questions and options, go to File > Download and choose Comma-separated values (.csv) to download the sheet as a .csv file.

4. Use an online tool like CSV to JSON Converter to convert your .csv file into a questions.json file.

Using Text Editors to Create the JSON File

1. Open your preferred text editor (Notepad, VS Code, Sublime Text, or any other code editor).

2. Create a JSON structure, starting with { "questions": [. Add each question as an object with the question and options fields.

3. Save the file with the name questions.json.

Example:

{
  "questions": [
    {
      "question": "Do you find interactive classes beneficial?",
      "options": ["Yes", "No"]
    },
    {
      "question": "What is your preferred learning method?",
      "options": ["Visual", "Auditory", "Kinesthetic"]
    }
  ]
}

Accessing the App from Other Devices (Windows Firewall)

If the app is not accessible from other devices, it may be blocked by your Windows firewall. To fix this:

  1. Go to Control Panel > System and Security > Windows Defender Firewall > Advanced Settings.
  2. Create an inbound rule to allow traffic on port 5000 (the default Flask port):
    • In the Windows Firewall with Advanced Security window, select Inbound Rules on the left.
    • Click on New Rule in the right-hand menu.
    • Select Port, then click Next.
    • Choose TCP and enter 5000 in the Specific local ports field.
    • Click Next, and then select Allow the connection.
    • Click Next again and select the appropriate network types (Domain, Private, or Public).
    • Give the rule a name (e.g., "Flask App - Port 5000") and click Finish.
